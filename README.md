##### `docker images [OPTIONS]`

| OPTION | INFO |
| :-: | :-: |
| -q | 只列出 image id |
| -a | 列出中間層 images |
| --digests | 列出摘要訊息 |
| --no-trunc | 列出完整訊息 |
